# Arena Cinema

## תיאור המערכת
Arena Cinema היא מערכת הזמנת כרטיסים לקולנוע הפועלת בצד הלקוח.

המערכת מאפשרת למשתמש:

- לעיין בסרטים בעמוד הבית  
- לצפות בפרטי סרט  
- לבחור הקרנה (מיקום, יום ושעה)  
- לבחור מושבים אינטראקטיבית ממפת האולם  
- לקבל סיכום הזמנה  

המערכת מדמה תהליך הזמנת הסרט מלך האריות.

---

## טכנולוגיות

המערכת פותחה באמצעות:

- HTML — מבנה העמודים  
- CSS — עיצוב רספונסיבי מותאם מובייל  
- JavaScript — לוגיקה אינטראקטיבית וניהול state  
- Bootstrap — רכיבי UI (קרוסלות בלבד)  
- LocalStorage — התמדה בצד לקוח  
- SVG — ציור מפת מושבים דינמית  

---

## סיכום פיצ'רים

### עמוד בית
- קרוסלות סרטים  
- ניווט בין עמודים  
- עיצוב רספונסיבי  

### עמוד הסרט – מלך האריות
- פרטי סרט מלאים  
- טריילר מוטמע  
- מעבר להזמנת כרטיסים  

### עמוד הזמנה (Booking)
- בחירת הקרנה (מיקום / יום / שעה)  
- מפת מושבים אינטראקטיבית ב-SVG  
- הבחנה בין מושבי VIP ורגילים  
- מניעת בחירת מושבים תפוסים  
- חישוב מחיר דינמי  
- ניהול state בצד לקוח  

### עמוד סיכום
- הצגת פרטי ההזמנה  
- פירוט מושבים  
- סה״כ לתשלום  
- טעינת נתונים מ-LocalStorage  

---

## רספונסיביות

המערכת מותאמת ל:
- Desktop  
- Tablet  
- Mobile  
- תפריט hamburger במובייל  

---

## הוראות הפעלה

1. לחצו על הקישור של האתר החי אשר יפנה אתכם למסך הבית  
2. לחיצה על התמונה של מלך האריות בקרוסלה תוביל לדף הסרט  
3. בדף הסרט לחצו על הכפתור "הזמנת כרטיסים"  
4. בחרו מיקום, יום ושעה ואז לחצו "הצג מפת מושבים"  
5. במובייל יש לגלול למטה כדי לראות את המפה  
6. בחרו מושבים (רגיל או VIP) ולאחר מכן לחצו לסיכום הזמנה  
7. בדף הסיכום יוצגו פרטי ההזמנה  
8. לחיצה על "לרכישת כרטיסים" תחזיר לעמוד הבית
